Sistema de Gestión para Plataforma de Streaming

Descripción

Este proyecto consiste en el desarrollo de un Sistema de Gestión para una Plataforma de Streaming desarrollado en el lenguaje de programación Go (Golang).

El sistema permite administrar usuarios, contenido multimedia y planes de suscripción mediante un menú interactivo en consola.

---

Objetivo

Desarrollar un sistema que permita aplicar los conocimientos adquiridos durante la asignatura, implementando programación estructurada y programación orientada a objetos, además de preparar la integración futura con servicios web y bases de datos.

---

Funcionalidades implementadas

- Inicio de sesión mediante usuario y contraseña.
- Registro de usuarios.
- Consulta de usuarios.
- Búsqueda de usuarios.
- Actualización de usuarios.
- Eliminación de usuarios.
- Registro de contenido multimedia.
- Consulta de contenido.
- Búsqueda de contenido.
- Actualización de contenido.
- Eliminación de contenido.
- Registro de planes de suscripción.
- Consulta de suscripciones.
- Estadísticas generales del sistema.
